MASKS - Manipal Application for SDG Keyword Suggestion

MASKS is an automated keyword-matching tool to ensure every research publication of MAHE is covered/mapped under the United Nation’s Sustainable Development Goals (SDGs).

Why MASKS

We are often unsure if the keywords that we publish are linked to any of the SDGs. Most of the research institutions/Universities have recommended that their researchers use a set of keywords that are likely to get covered under one or multiple SDGs. This, however, requires manual intervention and the efforts/time of the researchers. MASKS provides an immediate solution by automating the data-matching process from a large-scale authentic dataset for the benefit of researchers.

How it Works

MASKS is a simple and user-friendly tool that integrates all SDG-linked keywords from authentic sources like SciVal, which includes 18000+ unique records. This tool checks if the set of keywords provided is linked to any of the SDGs. More than a data-matching process, this tool provides additional options like spell-check and possible alternate suggestions based on partial matching.

  1. Input Data
    • MASKS can match up to 20 keywords at a time.
    • The keywords must be separated by any of the punctuation marks (comma, semicolon, colon).
    • Keywords separated by a simple space will be considered as a single keyword.
    • MASKS is case-insensitive and works in UPPER, lower, and Mixed cases.
  2. Output Information
    • Click the “basic search” button to match your keywords with the database.
    • Basic search results give you a direct A-A-matching of your keywords from the database.
    • The advanced search will provide a comprehensive list of possible keyword options from the database that partially match your keywords.
  3. Spell-check
    • In case of spelling errors, the application is built to indicate and suggest possible correct words.
  4. Add/Suggest keywords
    • The authors can also suggest keywords to be included in the database.

FAQs

  1. How to access this tool?
    The trial version of the tool is in the form of a web app that can be hosted on MAHE’s server and accessible to all users within MAHE.

  2. Will I be able to access this tool outside MAHE?
    As the tool is made available only to MAHE researchers, it is in the form of a web portal residing inside MAHE’s intranet. Hence, it will be accessible only through the intranet.

  3. What is the probability that keywords selected through MASKS get mapped under SDGs?
    The MASKS database is primarily sourced from authentic resources like SciVal; hence, there is a higher chance of keywords getting mapped under one or multiple SDGs.

  4. Is the dataset dynamic?
    The dataset is dynamic, and it will be updated regularly from authentic sources. In addition, end users can suggest the addition of potential keywords, which will be integrated into the database after verification. Click the “add/suggest keyword” button to update the keyword database. Basic information, including your name, institution, and email address, will be required while submitting your suggestions.

  5. What if I found no match for my keywords?
    MASKS is built to match the existing authentic keywords from the database. The author can suggest including those keywords in the database. Alternatively, the author can try different sets of keywords through the “Advanced Search” option. In addition, the next version of MASKS is expected to suggest keywords based on the abstract of the Manuscript.

  6. Does MASKS work offline?
    No. It works only online.

  7. Whether my login details or user data will be saved in the application?
    The authors need not login to access the MASKS. The user data will not be saved in any form.

  8. Whether theme-specific keywords are listed in MASKS?
    Keywords in MASKS are essentially pooled under each SDG category and there are no theme-specific keyword categories in MASKS. The platform remains the same for authors across different streams, including Health Sciences, Science and Technology, and Humanities.
